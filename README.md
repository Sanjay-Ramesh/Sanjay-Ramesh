## 🏢 Latest Project — WorkNest HRMS

> SaaS-style Workforce Management Platform — **Live Deployed**

[![Live Demo](https://img.shields.io/badge/Live%20Demo-work--nest--iota--nine.vercel.app-10b981)](https://work-nest-iota-nine.vercel.app)
[![GitHub](https://img.shields.io/badge/GitHub-WorkNest-0f172a)](https://github.com/Sanjay-Ramesh/WorkNest)
[![Tests](https://img.shields.io/badge/E2E%20Tests-13%2F13%20Passing-10b981)](https://playwright.dev)

**What it does:**
- 👤 Employee — apply leave, track balance and approval status
- 👥 Manager — approve/reject team leaves, own department only
- 📊 HR Admin — company-wide analytics, department breakdown
- 🔐 Role-based JWT security — each role sees only their data

**Stack:** Java 17 + Spring Boot + MongoDB Atlas, + React + Tailwind + Railway + Vercel
